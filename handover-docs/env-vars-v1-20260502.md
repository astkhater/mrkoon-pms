# Environment Variables — Mrkoon OKR Webapp
Date: 2026-05-02

## Frontend (Netlify build env)
| Var | Description | Required | Example |
|---|---|---|---|
| VITE_SUPABASE_URL | Supabase project URL | Yes | https://abc123.supabase.co |
| VITE_SUPABASE_ANON_KEY | Supabase anon public key | Yes | eyJhbG... |
| VITE_APP_NAME | App display name | Yes | Mrkoon Performance |
| VITE_APP_ENV | production / staging | Yes | production |
| VITE_DEFAULT_LANG | Default language | Yes | ar |

Set in Netlify → Site → Build & Deploy → Environment.

## Edge Function secrets (Supabase → Project → Edge Functions → Secrets)
| Var | Description | Required for | Example |
|---|---|---|---|
| SERVICE_ROLE_KEY | Supabase service-role key | Both functions | eyJhbG... (keep secret) |
| SUPABASE_URL | Same as frontend URL | Both | (auto-injected by Supabase) |
| CRM_SUPABASE_URL | Mrkoon CRM v15 project URL | crm-sync (live mode) | https://xyz789.supabase.co |
| CRM_SUPABASE_ANON_KEY | CRM anon key | crm-sync (live mode) | eyJhbG... |
| ERP_API_URL | ERP base URL | erp-sync (live mode) | https://erp.mrkoon.com/api |
| ERP_API_KEY | ERP API key | erp-sync (live mode) | (per ERP) |

Dormancy rule: if CRM_*  unset → crm-sync returns `mode:'dormant'`. Same for ERP.

## Local development (.env in frontend/)
Use `.env.example` as template. Never commit `.env`.
